# Archive

This directory stores historical material that is not part of the active V1
pipeline.

## Layout

- `datasets/legacy/`: old labeled examples, smoke-test data, and legacy ground truth.
- `datasets/bugsinpy/`: external BugsInPy material used during dataset exploration.
- `external_sources/`: copied upstream/project sources kept only for reference.
- `scripts/`: old helper scripts kept for traceability.
- `outputs/`: archived generated outputs.

The active V1 dataset lives in `examples/labeled/`.
The benchmark outputs used for the paper should be generated under `reports/`.
